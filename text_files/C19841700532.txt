 Hermed fremsendes ovennævnte meddelelse nr. 007.04 indeholdende  

 forskrifter for anbringelse af telefonmærker med pil på kantpæle.
 Meddelelse nr. 007.04 er godkendt af gruppen vedrørende eventuel  

 rationalisering af sagsbehandling inden for samarbejdsaftalen om bestyrelse  

 af hovedlandevejene.
 Yderligere eksemplarer af meddelelsen kan fås ved henvendelse til:

AAGE ANDERSEN overingeniør Meddelelse nr. 007.04 om færdselsregulering og afmærkning.
Anbringelse af  

 telefonmærker med pil på kantpæle.
I henhold til cirkulæreskrivelse af 14. juni 1978 j.nr.
A01-F0121-05 og  

 meddelelse nr. 007.01 af 14. juni 1979 om færdselsregulering og afmærkning  

 indkøber Vejdirektoratets samarbejdspartnere afmærkningsmateriel til  

 hovedlandevejene.
 For at hjælpe trafikanter, der på motorveje har fået motorstop og derfor  

 skal finde nærmeste nødtelefon, skal der på kantpælene i højre vejside  

 anbringes telefonmærker med pil visende til nærmeste nødtelefon, som angivet  

 på vedlagte bilag 1.
 Telefonmærket, der er af selvklæbende refleksmateriale, kan købes hos  

 leverandøren af kantpælene.
 Samarbejdspartnerne bedes påse at afmærkningen til stadighed er intakt.
 P.D.V.
